Portifólio Criado com objetivo de demonstrar um pouco de meu trabalho como programador e design.

Hoje irie contar um pouco de minha experiencia. Começei minha jornada de Programador a cerca de 2 anos em um projeto de desenvolvimento Mobile e atualmente estou focado em desenvolvimento no mundo de front-end, mas também gosto de fazer uma analise de dados usando SGBD SQL. Estou sempre Fazendo projetos e estudando para obter mais conhecimento possivel, para me tornar cada vez mais um programador completo.
Continue nesta página para mais detalhes:

Para começo no meu desenvolvimento Front-End começei com o HTML (Linguagem de Marcação de Hipertexto), o HTML é o componente base da web. Isso quer dizer que ele permite a construção de websites e a inserção de novos conteúdos, como imagens e vídeos, por meio dos hipertextos.

Logo em seguida começei a estlizar a página deixando-lá mais atrativa com o CSS, O CSS ajuda a editar, alinhar, remover e trabalhar no espaço entre elementos de uma página.

Fechando a seguencia da triplice de Front-End, temos o JavaScript . O JavaScript é uma linguagem de programação de alto nível criada, a princípio, para ser executada em navegadores e manipular comportamentos de páginas web.

Ao me aprofunda em JavaScript, com um tempo conheci uma fomosa Biblioteca do JavaScript o React. O React é um Biblioteca JavaScript criado pelo Facebook (atual Meta) que é usado para criar interfaces de usuário (UI) em aplicativos web. Ele é popular por ser fácil de usar, altamente flexível e escalável, e é usado por muitas empresas de tecnologia, incluindo o Facebook, Instagram e Airbnb.
